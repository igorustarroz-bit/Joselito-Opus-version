# Joselito — Design System

Librería de componentes del proyecto **Joselito** (WEB).
Construida con **React + Tailwind CSS + GSAP** y documentada en Storybook.

## Estructura

| Capa | Descripción |
| --- | --- |
| Tokens | Colores, tipografías, espaciados, breakpoints |
| Componentes | Botones, tags, inputs, tabs, toggles… |
| Módulos | Header, footer, heros (100% del viewport, grid 12 col) |
| Page Templates | Composición de módulos |

> El orden de trabajo es siempre Tokens → Componentes → Módulos → Templates.
> La estética la definen los Figmas, no Tailwind.
