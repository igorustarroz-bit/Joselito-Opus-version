# CardLink

## Intro
Tarjeta enlazable (Card Link): imagen + eyebrow + título + descripción + flecha. Toda la tarjeta es un enlace.

## Demo
<Canvas of={Stories.Default} />

## Anatomía
`AspectRatio` (imagen) + bloque de texto (eyebrow, Title, descripción) + "Leer más" con flecha.

## Subtemas por componente
Colores semánticos; el título vira a `--text-hover` en hover del grupo.

## Comportamiento
Hover desplaza la flecha y resalta el título; foco visible en toda la tarjeta.

## Variantes y tamaños
<Canvas of={Stories.Grid} />

## Tokens
`--bg-base`, `--text-base`, `--text-neutral-1`, `--text-hover`; tipografías `type-*`.

## Propiedades
<Controls of={Stories.Default} />

## Accesibilidad
Enlace único que envuelve toda la tarjeta, con foco de teclado visible.

## Componentes relacionados
CardProduct, AspectRatio, Title, Icon.
