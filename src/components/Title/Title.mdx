# Title

## Intro
Encabezado que aplica un estilo de título responsive (`type-title-1…8`) y color semántico, separando estilo visual de la etiqueta semántica (`as`).

## Demo
<Canvas of={Stories.Default} />

## Anatomía
Etiqueta de encabezado (`h1`–`h6`) con una clase `type-title-*` y `text-(--text-base)`.

## Comportamiento
El tamaño se adapta por breakpoint (definido en los tokens de tipografía).

## Variantes y tamaños
<Canvas of={Stories.Escala} />

## Tokens
`type-title-1…8`, color `--text-base`.

## Propiedades
<Controls of={Stories.Default} />

## Accesibilidad
Usa `as` para mantener una jerarquía correcta de encabezados independientemente del tamaño visual.

## Componentes relacionados
Tipografía (tokens), módulos Content / Title.
