# AspectRatio

## Intro

Mantiene una **proporción fija** de ancho:alto para su contenido (imágenes, vídeos,
mapas…), evitando saltos de layout mientras carga el medio. Origen: Figma →
*Foundations* → **Aspect Ratio**.

## Demo

<Canvas of={Stories.Default} />

## Anatomía

Contenedor con `aspect-ratio` CSS + una capa absoluta que estira el contenido
(`object-cover` en `img`/`video`). El ancho lo define el padre; el alto se calcula.

## Comportamiento

- El componente ocupa el 100% del ancho disponible; el alto se deriva del ratio.
- Las imágenes y vídeos se recortan (`object-cover`) para llenar el marco.

## Variantes y tamaños

<Canvas of={Stories.AllRatios} />

Ratios: `16:9`, `3:2`, `4:3`, `1:1`, `3:4`, `2:3`, `9:16`.

## Tokens

No usa tokens de color propios; el contenido define su apariencia.

<Controls of={Stories.Default} />

## Accesibilidad

El recorte es decorativo. Aporta `alt` significativo a las imágenes internas y
`aria-label`/subtítulos a los vídeos.

## Componentes relacionados

Se usa en tarjetas (Card Product, Card Link) y módulos con medios.
