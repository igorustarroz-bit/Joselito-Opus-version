# Checkbox

## Intro
Casilla de verificación (UI10) para selección múltiple. Tamaños S y M.

## Demo
<Canvas of={Stories.Default} />

## Anatomía
Input nativo oculto + caja visual (borde/relleno) + icono Check + etiqueta.

## Subtemas por componente
Colores semánticos (`--stroke-neutral-2`, `--bg-accent-base`, `--text-always-white`) → adaptable por tema.

## Comportamiento
Estados: sin marcar, marcado, hover, focus y disabled. Marca con el icono Check.

## Variantes y tamaños
<Canvas of={Stories.Estados} />
Tamaños: S (16px) y M (20px).

## Tokens
Borde `--stroke-neutral-2`; marcado `--bg-accent-base` + `--text-always-white`.

## Propiedades
<Controls of={Stories.Default} />

## Accesibilidad
Input `checkbox` nativo asociado por `label`; foco visible con `:focus-visible`.

## Componentes relacionados
Radio (UI10), Checkbox-List (UI08), Form (UI11).
