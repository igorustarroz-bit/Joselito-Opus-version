# UI10-Checkbox / Radio

Controles de selección de 20px. **Checkbox** cuadrado (esquinas 2px) que al marcarse se
rellena con el color de acento y muestra un check blanco. **Radio** circular con punto de
acento interior. Estados hover (borde acento), disabled y selected+disabled (gris).

## Tokens

Bordes `--stroke-neutral-1` (default), `--stroke-accent-1` (hover), `--stroke-neutral-3`
(disabled/selected). Relleno `--bg-accent-base` (marcado) / `--bg-neutral-3` (disabled).
Check en `--text-inverted`. Todo adapta por subtema.
