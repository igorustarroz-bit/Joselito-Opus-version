# UI09-Checkbox-Label

Casilla con **etiqueta**. Es el componente `Checkbox` con texto asociado. Tamaños **Medium**
(control 20px, etiqueta Body/03) y **Small** (control 16px, etiqueta Body/02). Estados:
default, hover, selected, **undefined** (indeterminado → minus), disabled y disabled selected.
La etiqueta usa `--text-neutral-1` (habilitada) / `--text-neutral-2` (deshabilitada).
