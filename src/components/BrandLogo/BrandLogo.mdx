# BrandLogo

## Intro

Logo de marca de Joselito (Figma → *Brand Assets* → Brand Logo). No es un icono, pero
funciona igual: SVG inline **recoloreable** con `currentColor`. Dos variantes:
**horizontal** (logotipo con el nombre) e **isotype** (el símbolo circular).

## Demo

<Canvas of={Stories.Default} />

## Anatomía

`<svg role="img">` con `viewBox` propio (300×200 horizontal, 200×200 isotipo) y trazos
en `currentColor`. Se dimensiona por `height`; el ancho se calcula por proporción.

## Color

Hereda el `color`; se recolorea con clases de texto/tokens y se adapta a los temas.

<Canvas of={Stories.Color} />

## Tokens

Sin color propio: usa `currentColor`. Aplica `--text-*` o el color de marca.

## Accesibilidad

`role="img"` con `aria-label`/`<title>` (por defecto "Joselito"). Úsalo dentro de un
enlace al inicio en cabeceras.

## Componentes relacionados

Icon; módulos Navigation, Menu, Footer (donde se usa como marca).
