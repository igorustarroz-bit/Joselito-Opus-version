# NavButton

## Intro
Botón de navegación (UI01) para menús y cabeceras. Estados default, hover, focus, selected y disabled.

## Demo
<Canvas of={Stories.Default} />

## Anatomía
Contenedor con texto (mayúsculas) e iconos opcionales; relleno/borde por estado.

## Subtemas por componente
Colores desde tokens `Button/Nav` (5 temas `data-theme`).

## Comportamiento
El item activo se marca con `selected`. `:focus-visible` muestra el foco.

## Variantes y tamaños
<Canvas of={Stories.Estados} />
Tamaño único (alto 56, texto `type-cta-m`).

## Tokens
`--btn-nav-<estado>-<fill|text|stroke|icon>`.

## Propiedades
<Controls of={Stories.Default} />

## Accesibilidad
`<button>` nativo; usa `aria-current` en el contenedor de navegación para el item activo.

## Componentes relacionados
Button (UI02), Tabs (UI05), módulos de Navigation.
