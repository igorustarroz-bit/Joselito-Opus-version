# Icon

## Intro

Sistema de iconos oficial (Figma → *Brand Assets* → Icons, set Phosphor).
Se renderizan como SVG inline y **heredan el color** con `currentColor`, por lo
que se recolorean con `color`/clases de texto. **Ningún componente debe usar un
icono fuera de esta lista.**

## Demo

<Canvas of={Stories.Default} />

## Anatomía

`<svg viewBox="0 0 32 32">` con trazos/rellenos en `currentColor`. El tamaño fija
`width`/`height`; el trazo escala proporcionalmente.

## Comportamiento

- Decorativo por defecto (`aria-hidden`). Con `title`, pasa a `role="img"` y expone el nombre accesible.
- Recoloreable: hereda el `color` del contenedor.

## Tokens

Sin color propio: usa `currentColor`. Aplica tokens de texto (`--text-*`) o de icono.

## Accesibilidad

Usa `title` cuando el icono transmita significado; déjalo vacío si es decorativo
(quedará `aria-hidden`).

## Componentes relacionados

Button, Tag, Input y la mayoría de componentes con iconografía.
