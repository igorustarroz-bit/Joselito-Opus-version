# UI06-Input

Campo de formulario. Tamaños **Big** (72px) y **Small** (56px); estados default, filled,
error, validated y disabled (hover/focus vía CSS). Botón de acción opcional a la derecha,
mensaje de ayuda opcional debajo. Rectangular.

## Tokens

Colores desde `--form-<estado>-<fill|stroke|text|message|...>` (grupo Forms de
Semantic-Color, por subtema). Tamaños y paddings desde `--form-l-*` / `--form-m-*`.

## Propiedades

| Prop | Tipo | Default | Descripción |
| --- | --- | --- | --- |
| `label` | `string` | `Label` | Placeholder |

| `state` | `default \| filled \| error \| validated \| disabled` | `default` | Estado |
| `message` | `string` | — | Mensaje de ayuda |

| `value` / `onChange` | — | — | Valor controlado |
