# Input

## Intro
Campo de texto (UI06) con etiqueta, mensaje de ayuda/error, iconos y estados. Tamaños M y L.

## Demo
<Canvas of={Stories.Default} />

## Anatomía
Etiqueta (label-out) + control (relleno, borde, icono, input) + mensaje inferior.

## Subtemas por componente
Colores desde tokens `Forms/*` por estado (5 temas `data-theme`).

## Comportamiento
Estados: default, hover, focus, filled, error, validated, disabled. `aria-invalid` en error.

## Variantes y tamaños
<Canvas of={Stories.Estados} />
Tamaños: M (56px) y L (72px).

## Tokens
`--form-<estado>-<fill|stroke|text|label-out|label-in|icon|message>`. Texto `type-input-*`.

## Propiedades
<Controls of={Stories.Default} />

## Accesibilidad
`label` asociado por `htmlFor`; `aria-describedby` para el mensaje y `aria-invalid` en error.

## Componentes relacionados
Listbox (UI07), Form (UI11), Checkbox/Radio (UI10).
