# Espaciado

## Intro

Dos escalas, ambas de Figma:

- **Fijos** (`--spacing-fx-*`, colección *Primitives*): no cambian con el breakpoint.
  Utilidades Tailwind: `p-fx-4`, `gap-fx-2`, `m-fx-8`…
- **Responsive** (`--space-0…18`, colección *Responsive*): cambian por breakpoint.
  Uso: `p-(--space-8)`, `gap-(--space-10)`.

También hay **radios** (`--radius-*`): `xxs xs s m` fijos y `l xl` responsive, más `rounded` y `none`.

> Cambia el **Breakpoint** en la barra superior para ver variar los espaciadores responsive.

## Demo

### Fijos

<div style={{ display: 'flex', flexDirection: 'column', gap: 6, margin: '12px 0' }}>
  {['fx-0','fx-1','fx-2','fx-3','fx-4','fx-5','fx-6','fx-7','fx-8','fx-9'].map((k) => (
    <div key={k} style={{ display: 'flex', alignItems: 'center', gap: 12 }}>

      <div style={{ height: 14, width: `var(--spacing-${k})`, background: 'var(--bg-accent-base)' }} />

  ))}
</div>

### Responsive

<div style={{ display: 'flex', flexDirection: 'column', gap: 6, margin: '12px 0' }}>
  {Array.from({length:19}, (_,i)=>`space-${i}`).map((k) => (
    <div key={k} style={{ display: 'flex', alignItems: 'center', gap: 12 }}>
      <code style={{ width: 72 }}>{k}</code>
      <div style={{ height: 14, width: `var(--${k})`, background: 'var(--bg-accent-subbrand-sec)' }} />
    </div>
  ))}
</div>

### Radios

<div style={{ display: 'flex', flexWrap: 'wrap', gap: 16, margin: '12px 0' }}>
  {['none','xxs','xs','s','m','l','xl','rounded'].map((k) => (
    <div key={k} style={{ textAlign: 'center' }}>
      <div style={{ width: 72, height: 72, background: 'var(--bg-neutral-2)', borderRadius: `var(--radius-${k})` }} />
      <code style={{ fontSize: 11 }}>{k}</code>
    </div>
  ))}
</div>

## Tokens

```tsx
<div className="p-fx-4 gap-fx-2" />          {/* fijos */}
<section className="p-(--space-12)" />        {/* responsive */}
<div className="rounded-m" />                 {/* radio fijo */}
<div style={{ borderRadius: 'var(--radius-xl)' }} /> {/* radio responsive */}
```
