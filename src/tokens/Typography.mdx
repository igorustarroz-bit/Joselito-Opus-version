# Tipografía

Estilos de texto **responsive**: el tamaño y el interlineado cambian por breakpoint
(datos de la colección Responsive). Las familias, pesos y letter-spacing vienen de las
primitivas. Cambia el **viewport** en la barra para ver la escala adaptarse.

- **Serif — SangBleu Kingdom:** `Title 2`–`Title 8`.
- **Sans — Euclid Circular B:** `Title 1`, `Body`, `CTA`, `Label`, `Input`.

Uso: aplica la clase `.ts-<estilo>` (p. ej. `.ts-title-4`, `.ts-body-3`).

## Tokens

Cada estilo expone dos custom properties responsive y una clase:

| Clase | Familia | Peso | Letter-spacing |
| --- | --- | --- | --- |
| `.ts-title-1` | Euclid Circular B | 400 | 0 |
| `.ts-title-2` … `.ts-title-7` | SangBleu Kingdom | 400 | -0.5px |
| `.ts-title-8` | SangBleu Kingdom | 400 | -3px |
| `.ts-body-1` … `.ts-body-6` | Euclid Circular B | 400 | 0 |
| `.ts-cta-xs` … `.ts-cta-xl` | Euclid Circular B | 400 | 0.5px |
| `.ts-label-1` · `.ts-label-2` | Euclid Circular B | 400 | 0 |
| `.ts-input-m-*` · `.ts-input-s-*` | Euclid Circular B | 400 | 0 |

Variables por estilo: `--ts-<estilo>-sz` (tamaño) y `--ts-<estilo>-lh` (interlineado),
responsive por breakpoint. Peso Medium (500) de SangBleu Kingdom disponible para énfasis.
