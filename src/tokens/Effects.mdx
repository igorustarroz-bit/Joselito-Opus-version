# Efectos

## Intro

**Elevaciones** (sombras) y **blur** de fondo, desde los *effect styles* de Figma.
Sombras en dos familias: `On-Light` y `On-Dark`. Utilidades Tailwind
`shadow-elevation-light-*` / `shadow-elevation-dark-*`; blur con `.blur-bg-s|m|l`.

## Demo

### Elevaciones (On-Light)

<div style={{ display: 'flex', flexWrap: 'wrap', gap: 32, padding: 24 }}>
  {['s','m','l','xl'].map((k) => (
    <div key={k} style={{ textAlign: 'center' }}>
      <div style={{ width: 96, height: 96, background: 'var(--bg-white)', borderRadius: 8, boxShadow: `var(--shadow-elevation-light-${k})` }} />
      <code style={{ fontSize: 11 }}>light-{k}</code>
    </div>
  ))}
</div>

### Elevaciones (On-Dark)

<div data-theme="dark-black-neutral" style={{ display: 'flex', flexWrap: 'wrap', gap: 32, padding: 24, background: 'var(--bg-base)' }}>
  {['s','m','l','xl'].map((k) => (
    <div key={k} style={{ textAlign: 'center' }}>
      <div style={{ width: 96, height: 96, background: 'var(--bg-neutral-1)', borderRadius: 8, boxShadow: `var(--shadow-elevation-dark-${k})` }} />
      <code style={{ fontSize: 11, color: 'var(--text-base)' }}>dark-{k}</code>
    </div>
  ))}
</div>

## Tokens

```tsx
<div className="shadow-elevation-light-m" />
<div className="blur-bg-s" />
```
