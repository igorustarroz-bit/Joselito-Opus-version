# Colores — Primitivas

## Intro

Las **primitivas** son la paleta cruda del sistema (colección **Primitives** de
Figma, modo único *Joselito*). **No se usan directamente** en componentes: son la
base sobre la que se construyen los [colores semánticos](/docs/tokens-colores-semánticos--docs).

Se exponen como utilidades Tailwind (`bg-primary-50`, `text-neutral-40`…) y como
variables CSS (`var(--color-primary-50)`).

## Tokens

- **Variable CSS:** `var(--color-<grupo>-<nivel>)` — p. ej. `var(--color-primary-50)`
- **Utilidad Tailwind:** `bg-<grupo>-<nivel>`, `text-…`, `border-…`

```tsx
// ❌ No usar primitivas directamente en componentes
<div className="bg-primary-50" />

// ✅ Usar el token semántico correspondiente
<div className="bg-(--bg-accent-base)" />
```
