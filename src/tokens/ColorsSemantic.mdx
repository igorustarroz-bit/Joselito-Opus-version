# Colores — Semánticos

## Intro

Los **colores semánticos** (colección **Semantic-Color** de Figma) son los que se
aplican **siempre** en componentes y módulos — nunca las primitivas directas.
Cada token apunta a una primitiva distinta según el **tema** (modo de Figma).

Hay **5 temas**, aplicados con el atributo `[data-theme]`. Un hijo puede
sobreescribir el tema del padre (p. ej. un footer oscuro con un bloque social claro).

Grupos: `--bg-*` (fondos), `--text-*` (textos), `--stroke-*` (bordes e iconos).

> Cambia el **Tema** en la barra de herramientas (arriba) para ver cómo mutan estos swatches.

## Tokens

```tsx
// Tailwind v4 (sintaxis de variable)
<div className="bg-(--bg-base) text-(--text-base) border border-(--stroke-neutral-2)" />

// Cambiar de tema (contenedor o hijo)
<footer data-theme="dark-black-neutral">
  <div data-theme="light-white">…bloque claro dentro del footer oscuro…</div>
</footer>
```
