# Responsive

Tokens que cambian por **breakpoint** (colecciĂłn `Responsive` de Figma). El proyecto es
**mobile-first**: la base es `XS` (390) y cada breakpoint sobreescribe hacia arriba.

Los breakpoints CSS coinciden con los de **Tailwind** (`@theme`) y con los **viewports de
Storybook** (barra de herramientas â†’ cambia el viewport para ver los tokens adaptarse).

> Usa el selector de **viewport** de la barra superior para comprobar cĂłmo cambian los
> spacers, gutters y esquinas en cada breakpoint.

### Layout fijo (no responsive)

Sizing de botones (`--btn-*`) y formularios (`--form-*`) es constante en todos los breakpoints
(p. ej. `--btn-l-size: 64`, `--btn-m-size: 56`, `--form-l-size: 72`, `--form-m-size: 56`).
