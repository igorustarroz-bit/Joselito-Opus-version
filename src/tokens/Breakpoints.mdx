# Breakpoints

## Intro

Los **breakpoints** definen los anchos de viewport en los que el diseño adapta su
composición. En Joselito provienen de Figma (colección **Responsive**,
grupo `Layout/Viewport-width`): cada breakpoint es un **modo** de las variables
responsive, y todos los tokens responsive (tipografías, espaciados, grid,
columnas) cambian de valor en función de él.

Son **mobile-first**: cada valor es un `min-width`. Un estilo aplicado en `m`
(768px) se mantiene en `lg`, `xl`, etc. salvo que se sobreescriba.

Se exponen de tres formas equivalentes, que deben coincidir siempre:

- **Tailwind** — variantes responsive: `xs:`, `sm:`, `m:`, `lg:`, `xl:`, `xxl:`, `xxxl:`
- **TS** — `src/tokens/breakpoints.ts` (fuente de verdad)
- **Storybook** — selector global «Breakpoint» en la barra superior

## Demo

Selecciona un breakpoint en la barra de herramientas (arriba, icono de móvil)
para encuadrar cualquier story a ese ancho. Anchos relativos:

<div style={{ display: 'flex', flexDirection: 'column', gap: 8, margin: '16px 0' }}>
  {breakpoints.map((b) => (
    <div key={b.name} style={{ display: 'flex', alignItems: 'center', gap: 12 }}>
      <code style={{ width: 56, flex: 'none' }}>{b.label}</code>
      <div
        style={{
          height: 22,
          width: `${(b.min / 1920) * 100}%`,
          background: 'color-mix(in srgb, currentColor 12%, transparent)',
          border: '1px solid color-mix(in srgb, currentColor 30%, transparent)',
          borderRadius: 4,
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'flex-end',
          paddingRight: 8,
          fontSize: 12,
          fontFamily: 'ui-monospace, monospace',
        }}
      >
        {b.min}px
      </div>
    </div>
  ))}
</div>

### Uso

```tsx
// Tailwind (mobile-first): 1 columna por defecto, 2 desde M, 4 desde XL
<div className="grid grid-cols-1 m:grid-cols-2 xl:grid-cols-4" />
```

```ts
// En JS/TS, desde la fuente de verdad
import { breakpoints, breakpointsMap } from '@/tokens/breakpoints';

breakpointsMap.m;   // 768
```

```css
/* CSS equivalente (min-width) */
@media (min-width: 768px) { /* M */ }
```
