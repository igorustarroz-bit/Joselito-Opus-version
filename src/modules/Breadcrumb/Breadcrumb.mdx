# Módulo · Navigation / Breadcrumb

Barra de navegación a **100% de ancho** con un enlace "volver" (flecha izquierda +
etiqueta, Body/03). Fondo base. Alto **60px** en mobile y **80px** en desktop. El padding
lateral usa el **wrapper del grid** (responsive). Cambia el viewport para verlo adaptarse.
