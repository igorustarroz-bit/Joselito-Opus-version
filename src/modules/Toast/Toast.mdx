# Toast (módulo)

## Intro
Notificación breve con icono, título, mensaje y cierre. Variantes info, success, error y warning.

## Demo

## Tokens
Fondos/textos de sistema: `--bg-success-1`, `--text-error`, etc. Cierre con ButtonIcon.

## Propiedades
<Controls of={Stories.Default} />

## Accesibilidad
`role="status"` + `aria-live="polite"`; el icono lleva `title` según la variante.

Icon, ButtonIcon.
