# Módulo · Navigation / Secondary menu

Fila de enlaces de categoría (Body/03) a 100% de ancho. En **desktop** va centrada; en
**mobile** se alinea a la izquierda con scroll horizontal. El ítem **activo** lleva un
subrayado negro. Cambia el viewport para verlo.
