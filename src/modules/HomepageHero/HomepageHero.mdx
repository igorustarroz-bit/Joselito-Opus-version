# HomepageHero (módulo)

## Intro

Hero de portada (Hero / Homepage hero) a pantalla completa: imagen/vídeo de fondo, overlay y contenido (título + CTA).

## Demo

## Comportamiento responsive

Alto mínimo configurable (`minHeight`). Contenido centrado o a la izquierda; el texto siempre en blanco sobre el overlay.
