# Módulo · Content / Text only

Bloque de texto a 100% de ancho. Variantes:
- **columns**: label (Body/06) + título (SangBleu Title/04) + cuerpo (Body/05) en una o dos columnas.
- **split**: título a la izquierda y cuerpo a la derecha (en desktop; apilado en mobile).

Padding lateral = wrapper del grid; contenido centrado (máx. 1320px).
